CreamInstaller: 自动DLC解锁安装器

此程序相当于一个Koaloader, SmokeAPI, ScreamAPI, Uplay R1 Unlocker, Uplay R2 Unlocker的组合工具, 用于解锁Steam, Epic Games, Uplay上游戏的 DLC(游戏拓展)

功能

  • 每当选择Steam游戏时，根据需要自动下载和安装SteamCMD。用于收集应用程序信息，例如名称，构建ID，列表，仓库等
  • 自动收集和缓存所有选定的 Steam 和 Epic 游戏及其 DLC 的信息。
  • 自动 DLL 安装和配置生成 Koaloader、SmokeAPI、ScreamAPI、Uplay R1 Unlocker 和 Uplay R2 Unlocker。
  • 自动卸载 Koaloader、CreamAPI、SmokeAPI、ScreamAPI、Uplay R1 Unlocker 和 Uplay R2 Unlocker 的 DLL 和配置。
  • 自动修复启动器（并通过右键单击上下文菜单“修复”选项手动修复）。*当您安装了CreamAPI，SmokeAPI或ScreamAPI时启动器更新时。

安装

从Github 仓库中下载 最新版本, 解压到任意位置, 然后运行 CreamInstaller.exe
若发现程序无法正常运行，请检查你的 .NET Runtime 版本是否为 7.0.8 及以上版本

运行

Step 1

勾选你所需要添加的游戏, 点击OK

Step 2

勾选需要解锁的游戏, 并且勾选Koaloader, 挂载于version.dll , 点击Generate and Install

Step 3

等待程序运行完成, 即可如图所示即可

Enjoy now!

原理

拥有DLC的Steam游戏会查询Steamworks API以获取所有可用 DLC 的列表来开始所有权验证. 这使得可以欺骗调用Steamworks API的游戏，使其认为您拥有所需的DLC.

提示

  1. 此程序绝对不适用于使用第三方DRM的游戏, 例如育碧, Rockstar等游戏. 此外, 此程序也不太可能在免费游戏中解锁任何东西, 因为它们通常将所有玩家数据存储在相应的游戏服务器上, 因此所有检查都是在服务器端进行的.
  2. 此程序不太可能与使用Anti-Cheat的游戏一起使用, 因为它们通常会检测到任何已被篡改的DLL / EXE. 有时可以禁用反作弊, 但这通常需要失去在线功能. 在相应的游戏主题中搜索有关如何禁用反作弊的更多信息.
  3. 有些游戏根本不使用任何DRM, 在这种情况下, 此程序是无用的. 您需要做的就是将DLC文件放在其他地方并将它们放入游戏文件夹中, 大多数GalGame都使用这种方法.
  4. 此程序仅适用于Windows 8及以上系统, 暂不支持使用Wine运行.
